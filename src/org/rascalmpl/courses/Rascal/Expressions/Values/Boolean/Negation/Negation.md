# Boolean Negation

.Synopsis
The _not_ operator on Boolean values.

.Index
!

.Syntax
`! _Exp_`

.Types

//

|         |            |
| --- | --- |
| `_Exp_` | `! _Exp_`  |
| `bool`   | `bool`     |


.Function

.Details

.Description
The _not_ operator on Boolean values defined as follows:

|          |            |
| --- | --- |
| `_Exp_`  | `! _Exp_`  |
| `true`   | `false`    |
| `false`  | `true`     |


.Examples
```rascal-shell
!true;
```
