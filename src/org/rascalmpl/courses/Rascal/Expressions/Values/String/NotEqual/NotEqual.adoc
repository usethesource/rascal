
[[String-NotEqual]]
# String NotEqual
:concept: Expressions/Values/String/NotEqual

.Synopsis
Not equal operator on string values.

.Index
!=

.Syntax
`_Exp_~1~ != _Exp_~2~`

.Types


|====
| `_Exp~1~_` | `_Exp~2~_` | `_Exp~1~_ != _Exp~2~_` 

| `str`     |  `str`    | `bool`               
|====

.Function

.Description
Yields `true` if both arguments are not identical and `false` otherwise.

.Examples
[source,rascal-shell]
----
rascal>"abc" != "defghi";
bool: true
rascal>"abc" != "abc";
bool: false
----
