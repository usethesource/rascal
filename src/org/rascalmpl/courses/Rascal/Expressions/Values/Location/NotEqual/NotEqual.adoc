# NotEqual
:concept: Expressions/Values/Location/NotEqual

.Synopsis
The not equal operator on location values.



.Syntax
`_Exp_~1~ != _Exp_~2~`

.Types

|====
| `_Exp~1~_` | `_Exp~2~_` | `_Exp~1~_ != _Exp~2~_` 

| `loc`     |  `loc`    | `bool`               
|====

.Function

.Description
Yields `true` if both arguments are not identical and `false` otherwise.
