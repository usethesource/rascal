[[Set-Equal]]
# Set Equal
:concept: Expressions/Values/Set/Equal

.Synopsis
Equal operator on set values.

.Syntax
`_Exp_~1~ == _Exp_~2~`

.Types


|====
| `_Exp~1~_`    |  `_Exp~2~_`    | `_Exp~1~_ == _Exp~2~_` 

| `set[_T~1~_]` |  `set[_T~2~_]` | `bool`               
|====

.Function

.Description
Yields `true` if both arguments are equal sets and `false` otherwise.

.Examples
[source,rascal-shell]
----
rascal>{1, 2, 3} == {3, 2, 1};
bool: true
rascal>{1, 2, 3} == {1, 2};
bool: false
----
