# Number Subtraction

.Synopsis
Subtract two numeric values.

.Index
-

.Syntax
`_Exp_~1~ - _Exp_~2~`

.Types


|            |             |                        |
| --- | --- | --- |
| `_Exp~1~_` |  `_Exp~2~_` | `_Exp~1~_ - _Exp~2~_`  |
| `int`     |  `int`     | `int`                |
| `int`     |  `real`    | `real`               |
| `real`    |  `real`    | `real`               |


.Function

.Details

.Description
Yields the numerical result of subtracting the value of _Exp_~2~ from the value of _Exp_~1~.

.Examples
```rascal-shell
13 - 12
13.5 - 12
12 - 13
12 - 13.5
```
