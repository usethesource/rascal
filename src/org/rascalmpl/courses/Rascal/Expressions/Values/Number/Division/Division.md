#### Synopsis

Division on numeric values.

#### Syntax

`Exp~1~ / Exp~2~`

#### Types


| `Exp~1~`  |  `Exp~2~` | `Exp~1~ / Exp~2~`   |
| --- | --- | --- |
| `int`      |  `int`     | `int`                 |
| `int`      |  `real`    | `real`                |
| `real`     |  `real`    | `real`                |


#### Function

#### Description

Yields the result of dividing the value of _Exp~1~_ by the value of _Exp~2~_.

#### Examples

```rascal-shell,error
12 / 3
10 / 3
12 / 3.0
10 / 3.0
12 / 0
```
