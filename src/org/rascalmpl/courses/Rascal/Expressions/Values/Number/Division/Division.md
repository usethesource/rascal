# Number Division

.Synopsis
Division on numeric values.

.Index
/

.Syntax
`_Exp_~1~ / _Exp_~2~`

.Types


| `_Exp~1~_`  |  `_Exp~2~_` | `_Exp~1~_ / _Exp~2~_`   |
| --- | --- | --- |
| `int`      |  `int`     | `int`                 |
| `int`      |  `real`    | `real`                |
| `real`     |  `real`    | `real`                |


.Function

.Details

.Description
Yields the result of dividing the value of _Exp~1~_ by the value of _Exp~2~_.

.Examples
```rascal-shell,error
12 / 3
10 / 3
12 / 3.0
10 / 3.0
12 / 0
```
