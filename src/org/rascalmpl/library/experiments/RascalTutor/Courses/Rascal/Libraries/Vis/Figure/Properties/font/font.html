Name Rascal/Libraries/Vis/Figure/Properties/font

Function
  1. FProperty font(str fname)
  2. FProperty font(computedStr cfname)
Description Sets the font to be used for text. See fontNames for a list of available fonts.
Examples 
t1 = text("Rascal", font("Serif"), fontSize(20), fontColor("Blue"));
t2 = text("Rascal", font("Monospaced"), fontSize(20), fontColor("Red"));
render(hcat([t1, t2], hgap(10), fontSize(20)));
