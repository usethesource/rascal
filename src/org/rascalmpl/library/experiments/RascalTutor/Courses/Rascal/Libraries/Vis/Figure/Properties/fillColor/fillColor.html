Name Rascal/Libraries/Vis/Figure/Properties/fillColor

Function
  1. FProperty fillColor(Color c)
  2. FProperty fillColor(computedColor cc)
  3. FProperty fillColor(str cname)
Description Set the color used to fill the inside of figures:
  1. Set the fill color to c.
  2. Set the fill color to the value of the callback function cc.
  3. Set the fill color to the color with name cname.
Also see ColorModel.
Examples 
b1 = box([size(50), fillColor("Red")]);
b2 = box([size(50), fillColor("Blue")]);
b3 = box([size(50), fillColor("Green")]);
b4 = box([size(50), fillColor("Purple")]);
boxes = [b1, b2, b3, b4];
render(hcat(boxes));
